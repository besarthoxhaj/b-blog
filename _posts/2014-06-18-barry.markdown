Below is a step-by-step guide about how to set up a development area in Nitrous.io using Github pages ..

1.Look at what you can do .. firstly set-up a Github (public) account and create a repository (bunch of files where you can store the docs/content for your website..). In this example we set-up a repository called 'ettore90/blog' for our web content.

<center>
  <img src="http://i59.tinypic.com/bh12lx.jpg" alt="..." class="img-thumbnail" style="width:70%;height:auto;margin:10px 10px 30px 10px;">
</center>

2.The folders in your newly created Github repository will look something like this below..

<center>
  <img src="http://i59.tinypic.com/w4bk3.jpg" alt="..." class="img-thumbnail" style="width:70%;height:auto;margin: 0 -10px 30px;">
</center>

3.Once your Github repository has been created you can 'clone' (copy+paste) the repository folders/files into the Nitrous development area by copying your Gitbub repository url, easy'huh :)

4.Now would be a good time to create an account/profile in Nitrous.io. Once in Nitrous click on the 'new box' tab and then click the 'create box' tab (see example below)

<center>
  <img src="http://i57.tinypic.com/4kb2tt.jpg" alt="..." class="img-thumbnail" style="width:70%;height:auto;margin: 0 -10px 30px;">
</center>

5.You'll know your in your development area when you see the page below the most important part in this step is getting familar with the command-line, this is where you will clone (cut+paste) your Github repository in effect creating a copy of your website files.. Please type in the command-line git clone "github url" and hit return!

<center>
  <img src="http://i59.tinypic.com/2utm6om.jpg" alt="..." class="img-thumbnail" style="width:70%;height:auto;margin: 0 -10px 30px;">
</center>

6.Enjoy building your webiste in your own 'dev' area .. :) but read step 7 first!
